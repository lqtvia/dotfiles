## keybinds

super = windows key

* `super + enter` - terminal (kitty)
* `super + space` - app launcher (rofi)
* `super + w` - firefox
* `super + e` - yazi
* `super + m` - music (rmpc)
* `super + x` - power menu
* `super + c` - close window
* `super + v` - toggle float
* `super + shift + v` - clipboard
* `super + f` - maximize
* `super + shift + f` - fullscreen
* `super + s` - btop
* `super + n` - notifications
* `super + p` - localsend
* `super + tab` - overview
* `super + 1-0` - workspaces
* `super + arrows` - move focus
* `super + shift + arrows` - move window
* `super + mouse scroll` - switch workspace
* `super + shift + s` - change wallpaper
* `super + shift + a` - dark/light mode toggle
* `super + l` - lock brightness
* `super + \` - scratchpad terminal
* `print` - screenshot area
* `shift + print` - screenshot whole screen
* `3 fingers down` - close window
* `3 fingers up` - fullscreen

look at `hypr/.config/hypr/binds.lua` for all the extra stuff
